# Nowruz Digital Exhibition

An immersive digital exhibition about **Persian Nowruz**, designed as a polished interactive website for GitHub Pages.

This project explores:

- the exact astronomical moment of the **March equinox**
- the ritual timeline of **Nowruz**
- the symbolic language of the **Haft-Seen**
- the sonic atmosphere of **Persian classical music**
- the culinary culture surrounding the Persian New Year

## Project Overview

This website is built as a **single-file digital exhibition** using HTML, CSS, and vanilla JavaScript.

It combines:
- scroll-reactive Persian geometric background patterns
- animated section reveals
- Persian typography
- interactive exhibition panels
- embedded YouTube music selections
- a responsive layout for desktop and mobile

## Features

- Immersive hero section with English and Persian typography
- Animated Persian architectural pattern system that changes while scrolling
- Astronomical section on the March equinox
- Ritual timeline of Nowruz ceremonies
- Interactive Haft-Seen section
- Expanded Persian music gallery with embedded videos
- Interactive culinary section in exhibition format
- Responsive design for mobile and desktop
- Single-file architecture for simple deployment

## Deployment

This project is intended to be deployed with **GitHub Pages** from the repository root.

Recommended publishing source:
- Branch: `main`
- Folder: `/ (root)`

## GitHub Pages Setup

1. Create a new GitHub repository
2. Upload all files in this project
3. Go to **Settings → Pages**
4. Under **Build and deployment**, choose:
   - **Source:** Deploy from a branch
   - **Branch:** `main`
   - **Folder:** `/ (root)`
5. Save and wait for deployment
