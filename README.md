# Nowruz Digital Exhibition

An immersive digital exhibition about **Persian Nowruz**, designed as a polished interactive website for GitHub Pages.

This project explores:

- the exact astronomical moment of the **March equinox**
- the ritual timeline of **Nowruz**
- the symbolic language of the **Haft-Seen**
- the sonic atmosphere of **Persian classical music**
- the culinary culture surrounding the Persian New Year
